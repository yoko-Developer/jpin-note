`*` で三角を作る

1行目：改行だけ（なにもなし）

2行目：*

3行目：**

4行目：***

```
package date20250723;

public class Triangle {
    public static void main(String[] args) {

        // 横を制御する
        for (int i = 0; i <= 3; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}
```

✅ System.out.print("*");

    内側の for の中で、条件を満たしたときだけ実行される


✅ System.out.println();

    外側の for の中で、内側が終わったあとに毎回実行される


1週目: i = 0
- i <= 3 : ✅
- j = 1
- j <= i → 1 <= 0 ❌ → 内側のループには入らないから``System.out.print("*")` は実行されない❌ 
- `System.out.println();` → ✅
⭐️ 表示：何も出ない + 改行

2週目: i = 1
- j = 1; j <= 1 → ✅
  - `System.out.print("*");` → `*` 1個表示！
- j = 2; j <= 1 → ❌
- System.out.println(); → 改行
⭐️ 表示: `*` (1個) + 改行

3週目: i = 2
- j = 1; j <= 2 → ✅ → `*`
- j = 2; j <= 2 → ✅ → `*`
- j = 3; j <= 2 → ❌
- `System.out.println();` → 改行
⭐️ 表示: `* *` (2個) + 改行

4週目: i = 3
- j = 1; j <= 3 → ✅ → `*`
- j = 2; j <= 3 → ✅ → *
- j = 3; j <= 3 → ✅ → *
- j = 4; j <= 3 → ❌
- System.out.println(); → 改行
⭐️ 表示: `* * *` (3個) + 改行
